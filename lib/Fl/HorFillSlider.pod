class 'Fl::HorFillSlider';
isa 'Fl::Slider';
include 'FL/Fl_Hor_Fill_Slider.H';
widget_type 'Fl_Hor_Fill_Slider';

=pod

=head1 NAME

Fl::HorFillSlider - Filled Horizontal Slider

=head1 Description

The Fl::HorFillSlider draws a filled horizontal slider, useful as a progress
or value meter.

=head1 Methods

Fl::HorFillSlider inherits from Fl::Slider, Fl::Valuator and Fl::Widget. On
top of that, it exposes the following methods...

=head2 new(...)

    my $val_a = Fl::HorFillSlider->new(0, 0, 250, 500, 'Important Stuff');
    my $val_b = Fl::HorFillSlider->new(0, 0, 250, 500);

The constructor creates a new widget using the given position, size, and
label.

The default boxtype is FL_DOWN_BOX.

The destructor removes the widget.

=cut

};
